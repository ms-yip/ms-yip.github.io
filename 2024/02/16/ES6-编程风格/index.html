ES6 编程风格

发表于2024-02-16|更新于2024-09-26|ES6
|字数总计:585|阅读时长:1分钟|阅读量:

块级作用域

  • 使用 let 取代 var
    • 避免全局变量
    • 避免变量提升
  • 优先使用 const
    • 适合全局常量
    • 提高程序运行效率
    • 防止无意修改变量值

字符串

  • 静态字符串使用单引号或反引号
  • 动态字符串使用反引号

解构赋值

  • 数组成员赋值优先使用解构赋值
  • 函数参数使用对象解构赋值
  • 函数返回多个值使用对象解构赋值

对象

  • 单行定义对象，最后一个成员不以逗号结尾
  • 多行定义对象，最后一个成员以逗号结尾
  • 尽量静态化对象
  • 动态属性名使用属性表达式定义
  • 对象的属性和方法使用简洁表达法

数组

  • 使用扩展运算符拷贝数组
  • 使用 Array.from 方法将类似数组的对象转为数组

函数

  • 立即执行函数使用箭头函数
  • 匿名函数参数使用箭头函数
  • 箭头函数取代 Function.prototype.bind
  • 简单单行函数使用箭头函数
  • 配置项集中在一个对象，放在最后一个参数
  • 使用 rest 运算符代替 arguments
  • 使用默认值语法设置函数参数的默认值

Map 结构

  • 区分 Object 和 Map
    • 需要 key: value 数据结构时使用 Map

Class

  • 用 Class 取代 prototype 操作
  • 使用 extends 实现继承

模块

  • 使用 import 取代 require()
  • 使用 export 取代 module.exports
  • 默认输出一个函数时函数名首字母小写
  • 默认输出一个对象时对象名首字母大写

ESLint 的使用

  • 安装 ESLint
  • 安装 Airbnb 语法规则和插件
  • 配置 .eslintrc 文件
  • 使用 ESLint 检查代码

检查漏点

  • 块级作用域：强调避免变量提升，强调块级作用域
  • 字符串：清晰区分静态和动态字符串的引号使用
  • 解构赋值：优先使用解构赋值，函数参数解构，返回值解构
  • 对象：静态化，动态属性表达式，简洁表达法
  • 数组：扩展运算符，Array.from 转换
  • 函数：箭头函数，配置对象，rest 运算符，默认值语法
  • Map 结构：Map 的使用及遍历机制
  • Class：类和继承的使用
  • 模块：import 和 export 的使用规范
  • ESLint：安装和配置，使用规则
