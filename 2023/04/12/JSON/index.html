JSON

XML传输结构化数据的事实标准

JSON 一种数据格式，非从属于javascript。可以使用eval()解析、解释并返回javascript对象和数组

JSON语法可以表示的类型：

简单值：字符串（JSON字符串必须用双引号）、数值、布尔型、null；不支持undefined

对象：键值对，值可以是简单值也可以是复杂数据类型的值（没有变量的概念，不需要末尾分号，键要加双引号）

数组：有序列表 可以是任何类型：简单值、对象、数组（没有变量和分号）

序列化：（所有函数和原型成员会被忽略，值为undefined的属性会被跳过）

  1. JSON.stringify(js对象,[ [ ] | function(key, value){}], [string | nunber)

参数二： 过滤器(数组)或者是过滤规则函数

参数三：是否保留缩进和换行，最大值是10，或者字符长度最大是10；

​ 如果缩进参数是一个字符串而非数值，那么 JSON 字符串中就会使用这个字符串而不是空格来缩进。

  1. 定义对象的toJSON方法

序列化顺序：

有toJSON 方法 并能取得有效值调用此方法，否则返回对象本身。

解析：

JSON.parse( js对象,[ function(key, value){} ] )
